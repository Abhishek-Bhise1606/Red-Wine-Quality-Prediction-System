# 🍷 Red Wine Quality Prediction System

A Machine Learning project that predicts the quality of red wine based on various physicochemical properties using Python and Scikit-learn.

---

## 📌 Project Overview

The goal of this project is to build a predictive machine learning model that can determine the quality of red wine using different chemical features such as acidity, sugar content, alcohol percentage, pH value, and more.

This project demonstrates:

* Data preprocessing
* Exploratory Data Analysis (EDA)
* Feature engineering
* Machine Learning model training
* Model evaluation
* Prediction system development

---

## 📂 Dataset Information

The dataset contains different physicochemical properties of red wine samples.

### Features Used

* Fixed Acidity
* Volatile Acidity
* Citric Acid
* Residual Sugar
* Chlorides
* Free Sulfur Dioxide
* Total Sulfur Dioxide
* Density
* pH
* Sulphates
* Alcohol

### Target Variable

* **Quality** → Wine quality score
